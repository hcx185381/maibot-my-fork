### 4. 配置设置 (5:30-7:30)

**画面：**
```
[5:30-6:00] 打开配置文件
  找到 docker-config/mmc/bot_config.toml
  用记事本打开

[6:00-6:40] 修改人设
  找到 [nickname] 部分
  修改昵称、人设、说话风格

[6:40-7:30] 配置活跃度
  找到 [talk_value_rules]
  解释活跃度数值
  展示如何配置不同群组
```

**台词：**
```
现在我们来配置机器人的个性。

打开项目文件夹，
进入 docker-config/mmc，
找到 bot_config.toml，
用记事本打开。

我们先设置机器人的人设。

看到这里，[nickname] 部分，
nickname 就是机器人的名字，
我们可以改成"小喵"。

personality 是人设描述，
比如"一个可爱的女大学生，性格活泼"。

reply_style 是说话风格，
可以让机器人每句话都加"呢、啊、呀"等语气词。

接下来配置群聊活跃度。

找到 talk_value_rules，
这里的 value 数值决定活跃度：
- 0.8 表示很活跃，经常主动聊天
- 0.05 表示几乎安静，很少主动说话

比如，在电协群设置 0.8（活跃），
在其他群设置 0.05（安静）。

配置完成后，
记得按 Ctrl+S 保存，
然后重启容器使配置生效。
```

**备注：**
- 配置文件特写
- 数值用字幕标注
- 保存和重启步骤

## 📹 录制注意事项

### 技术要求
- **分辨率：** 1920x1080 或更高
- **帧率：** 30fps
- **音频：** 清晰无杂音
- **字幕：** 添加中文字幕

### 屏幕录制
推荐使用：
- OBS Studio（免费、专业）
- Bandicam（简单易用）
- Camtasia（功能强大）

### 录制技巧
1. **字体设置：** 放大系统字体到 125%
2. **窗口大小：** 窗口不要最大化，方便观看
3. **鼠标轨迹：** 开启鼠标高亮
4. **快捷键：** 提前熟悉快捷键操作

- **简单：** 剪映（免费，手机电脑都能用）
- **专业：** DaVinci Resolve（免费专业级）
- **入门：** Windows Movie Maker（简单）
