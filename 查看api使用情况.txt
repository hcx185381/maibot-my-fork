  2️⃣ 如何通过命令行查看Docker日志

  📋 基础命令（在MaiBot目录下打开命令行）

  方法1：查看最近50行日志
  docker-compose logs --tail=50 core

  方法2：实时查看日志（持续更新）
  docker-compose logs -f core
  按 Ctrl+C 停止查看

  方法3：查看最近100行并持续监控
  docker-compose logs -f --tail=100 core

  ---
  🔍 查看特定内容的日志

  查看API调用统计：
  docker-compose logs core | grep "模型名称"

  查看错误信息：
  docker-compose logs core | grep -i "error"

  查看表情包相关：
  docker-compose logs core | grep "表情包"

  查看最近1小时的日志：
  docker-compose logs --since=1h core
